在我们运行模型的时候，首先跳出来的界面是进行禁词输入，根据界面提示来添加禁词，添加完成后点击save进行保存。
我们诗词生成器的应用有三种模式，分别是：
随机生成。
根据提示语句生成藏头诗。
以提示语句开头生成完整诗词。
